💬 Chit-Chat Web App
A modern, real-time chatting web application built using HTML, CSS, JavaScript, PHP, and JSON — designed to deliver a smooth and interactive messaging experience.
🚀 Overview

Chit-Chat is a lightweight yet powerful messaging platform where users can register, log in, and communicate in real-time. The project focuses on simplicity, performance, and user-friendly design while implementing core web development concepts.

✨ Features
🔐 User Authentication (Register/Login/Logout)
💬 Real-time Messaging System
👤 User Profiles with Profile Pictures
🔍 Search Users by Username
🟢 Online/Offline Status Indicator
📁 JSON-based Database (No MySQL required)
🎥 Audio/Video Calling (WebRTC Integration)
📷 Media Sharing (Images, Files)
🌙 Dark Mode UI
🔔 Notifications System
⚡ Single Page Application (SPA) Experience
🛠️ Tech Stack
Technology	Purpose
HTML	Structure
CSS / Tailwind CSS	Styling & UI
JavaScript	Frontend Logic
PHP	Backend Processing
MOngoDb	Database Storage
WebRTC	Audio/Video Calls
